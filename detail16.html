# beautifulsoup مقدمه


کتابخانه beautifulsoup یک کتابخانه پایتون است که بمنظور استخراج داده از فایل‌های html و xml مورد استفاده قرار می گیرد. این کتایخانه صفحات مورد نظر خودرا بصورت یک درخت تجزیه می‌کند. درخت تجزیه این امکان را برای برنامه ایجاد می‌کند، که هرگونه دسترسی به عناصر صفحه html با سرعت بیشتری امکان‌پذیر گردد.

نصب beautifulsoup
  1. وارد محیط پروژه خود شوید با استفاده از ترمینال
  2. pip install beautifulsoup4

# دستورات این کتابخانه


دستور توضیحات
BeautifulSoup(html_string, parser) این دستور یک شیء BeautifulSoup از یک رشته HTML ایجاد می کند. پارامتر parser نوع تجزیه کننده HTML را تعیین می کند. مقدار پیش فرض "html.parser" است.
soup.find_all(name) این دستور تمام عناصر HTML با نام مشخص شده را برمی گرداند. پارامتر name نام عنصر HTML را تعیین می کند.
soup.find(name) این دستور اولین عنصر HTML با نام مشخص شده را برمی گرداند.
soup.select(selector) این دستور تمام عناصر HTML با انتخابگر CSS مشخص شده را برمی گرداند. پارامتر selector انتخابگر CSS را تعیین می کند.
soup.select_one(selector) این دستور اولین عنصر HTML با انتخابگر CSS مشخص شده را برمی گرداند.
soup.text این دستور متن داخل عنصر HTML را برمی گرداند.
soup.attrs این دستور یک دیکشنری از ویژگی های عنصر HTML را برمی گرداند.

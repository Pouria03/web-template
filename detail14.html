# مراحل نصب لینوکس


1. دانلود ایمیج ISO: ابتدا، ایمیج ISO مربوط به توزیع مورد نظر خود را از وب‌سایت رسمی آن دانلود کنید. برای مثال، اگر می‌خواهید Ubuntu را نصب کنید، می‌توانید ایمیج ISO را از وب‌سایت رسمی Ubuntu دانلود کنید.

2. ساخت USB بوتیبل: بعد از دانلود ایمیج ISO، باید یک USB بوتیبل بسازید. برای این کار، می‌توانید از نرم‌افزار‌هایی مانند Rufus یا Etcher استفاده کنید.

3. تنظیم BIOS یا UEFI: بعد از ساخت USB بوتیبل، باید کامپیوتر خود را ری‌استارت کنید و وارد تنظیمات BIOS یا UEFI شوید. در این قسمت، باید ترتیب بوت را تغییر دهید تا کامپیوتر از روی USB بوت شود.

4. نصب لینوکس: در این مرحله، باید از روی USB بوت کنید و دستورالعمل‌های مربوط به نصب لینوکس را دنبال کنید. این شامل انتخاب زبان، تنظیمات شبکه، انتخاب نوع نصب (مانند نصب کنار ویندوز یا پاک کردن همه چیز و نصب لینوکس)، و انتخاب محل نصب است.

5. راه‌اندازی اولیه: پس از نصب، کامپیوتر خود را ری‌استارت کنید و از روی هارد دیسک بوت کنید. سپس، باید راه‌اندازی اولیه را انجام دهید، از جمله تنظیمات کاربر و زمان.

لطفاً توجه داشته باشید که این فقط یک راهنمای ساده است و بسته به توزیع لینوکس و سخت‌افزار شما، ممکن است نیاز به مراحل اضافی داشته باشید.

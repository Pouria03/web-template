قوانین نامگذاری در پایتون

نام گذاری متغیرها، توابع، کلاس ها و سایر عناصر در پایتون باید مطابق با قوانین خاصی باشد. این قوانین به خوانایی و درک بهتر کد کمک می کنند.

قوانین نام گذاری در پایتون عبارتند از:

  • نام ها باید از حروف الفبای انگلیسی، اعداد و کاراکترهای زیر استفاده کنند: * زیرخط (_) * نقطه (.)
  • نام ها نباید با اعداد شروع شوند.
  • نام ها نباید از کلمات کلیدی پایتون استفاده کنند.
  • نام ها باید مختصر و واضح باشند. 

# متغیر
my_variable = 10

# تابع
def my_function():
  pass

# کلاس
class MyClass:
  pass


# متغیر
10_variable = 10

# تابع
def function():
  pass

# کلاس
class Class:
  pass

توصیه ها برای نام گذاری در پایتون:

  • از نام های معنادار استفاده کنید.
  • از حروف کوچک استفاده کنید.
  • از زیرخط برای جدا کردن کلمات در یک نام استفاده کنید. 

# variable
total_score = 100

# function
calculate_area()

# class
Rectangle


# variable
my_variable = 10

# function
my_function()

# class
my_class


# variable
my_variable_name = 10

# function
my_function_name()

# class
my_class_name

توصیه های اضافی برای نام گذاری در پایتون:

  • از نام های یک کلمه ای برای متغیرهای ساده استفاده کنید.
  • از نام های چند کلمه ای برای متغیرها، توابع و کلاس های پیچیده تر استفاده کنید.
  • از زیرخط برای جدا کردن کلمات در یک نام استفاده کنید.
  • از نام های معنادار استفاده کنید تا کد خود را برای خوانندگان آسان تر کنید.

با رعایت قوانین نام گذاری در پایتون می توانید کد خود را خواناتر و قابل درک تر کنید.
